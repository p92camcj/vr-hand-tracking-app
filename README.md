# Neon VR Hand Studio (Production)

This is the stable, publicly accessible version of the Neon VR Hand Studio.

## 🚀 Live Demo
You can access the application here: [https://p92camcj.github.io/vr-hand-tracking-app/](https://p92camcj.github.io/vr-hand-tracking-app/)

## 🛠️ Features
- Real-time gesture recognition.
- Air-painting with glowing neon effects.
- HUD with custom color palette.
- Multi-hand support.

## 🛡️ Security & Privacy
This repository only contains stable release builds. Experimental code and development history are maintained in a private environment.
